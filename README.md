# E-Commerce Sales Dashboard

## Project Overview
This repo contains datasets and a dashboard for analyzing e-commerce sales. The goal is to track performance across categories, customers, states, and payment modes, and to highlight trends in profit and quantity sold.

The dashboard is built from two CSV files:
- **Details.csv** → Transaction-level data (amount, profit, quantity, category, sub-category, payment mode).
- **Orders.csv** → Order metadata (date, customer, state, city).

By combining these, we can answer questions like:
- Which states generate the most profit?
- Which product sub-categories are most profitable?
- How do customers prefer to pay?
- What months show peak performance?

## Features
- KPIs: Total Sales, Profit, Quantity, Avg Order Value
- Charts:
  - Profit by State
  - Quantity by Payment Mode
  - Profit by Sub-Category
  - Amount by Customer
  - Category-wise Quantity
  - Monthly Profit Trends
- Filters: Quarter, Category, Payment Mode

## Tech Stack
- **Data**: CSV files
- **Processing**: Power BI,Excel
- **Visualization**: Bar charts,donut chart, slicer,KPI card
